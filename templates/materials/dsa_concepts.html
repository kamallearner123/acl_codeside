DSA Concepts

DSA Fundamentals (12hrs)
  • Algorithm Analysis: Big-O, Big-Ω, Big-Θ notations for loops and recursion. Best, average, and worst case analysis.
  • Arrays & Strings: Indexing, searching, sorting (Bubble, Insertion, Selection), palindrome checks, sliding window (max subarray, longest unique substring).
  • Linked Lists: Singly, doubly, and circular lists. Insertions, deletions, reversal (iterative/recursive), Floyd's cycle detection.
  • Stacks & Queues: Implemented with arrays/linked lists. Parentheses balancing, LRU cache.
  • Recursion: Tail vs non-tail recursion, recursion tree, backtracking basics.
  • Hashing: Hash tables, collision resolution, applications in lookups and caches.
  • Searching: Linear search, binary search, search in rotated arrays.
  • Sorting: Merge sort, quick sort, stability, in-place vs out-of-place.
  • Practice Problems: Coding exercises for each topic, including edge cases and optimizations.
Tip: Each section includes code snippets, visualizations, and real-world applications to reinforce concepts.
