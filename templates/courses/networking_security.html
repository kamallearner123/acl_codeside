Course Curriculum

1

TCP/IP Protocols with Programming

5 hours
  • TCP/IP Stack Fundamentals
    OSI model, packet structure, IPv4/IPv6, subnetting, and routing protocols
  • Socket Programming in C++
    TCP/UDP client-server applications, Berkeley sockets API, and network programming patterns Hands-on
  • Network Programming with Rust
    Tokio async networking, std::net module, and safe concurrent network applications
  • Protocol Analysis and Wireshark
    Packet capture, traffic analysis, and network troubleshooting techniques Analysis
  • Advanced Protocols Implementation
    HTTP/HTTPS, FTP, SMTP, DNS custom implementations and protocol design Hands-on
2

SSL/TLS and Digital Certificates

4 hours
  • SSL/TLS Protocol Deep Dive
    Handshake process, cipher suites, TLS 1.2 vs 1.3, and perfect forward secrecy
  • Digital Certificates and X.509
    Certificate structure, validation chains, CRL, OCSP, and certificate pinning Crypto
  • Cryptographic Algorithms
    RSA, ECC, AES, SHA family, and practical cryptography implementation
  • TLS Implementation in Applications
    OpenSSL programming, secure socket creation, and certificate validation in C++ and Rust Hands-on
  • SSL/TLS Security Analysis
    Common vulnerabilities, SSL Labs testing, and security assessment tools Security
3

Public Key Infrastructure (PKI)

4 hours
  • PKI Architecture and Components
    Certificate Authorities, Registration Authorities, certificate lifecycle management
  • Building a Certificate Authority
    OpenSSL CA setup, intermediate CAs, certificate signing, and revocation lists Hands-on
  • Digital Identity and Authentication
    Client certificates, smart cards, hardware security modules (HSMs), and multi-factor authentication
  • Enterprise PKI Deployment
    Active Directory Certificate Services, automated enrollment, and certificate templates Enterprise
  • Modern PKI Applications
    Code signing, device certificates, IoT PKI, and blockchain certificate management
4

Network Security and Firewalls

4 hours
  • Firewall Technologies and Types
    Packet filtering, stateful inspection, application-layer firewalls, and next-generation firewalls
  • iptables and Advanced Filtering
    Linux firewall configuration, NAT, port forwarding, and custom rule chains Hands-on
  • Network Reconnaissance with Nmap
    Port scanning, OS fingerprinting, service detection, and vulnerability scanning techniques Ethical Hacking
  • Network Monitoring and Analysis
    Traffic flow analysis, bandwidth monitoring, and network security monitoring (NSM)
  • VPN and Secure Remote Access
    IPSec, OpenVPN, WireGuard implementation and secure tunnel configuration VPN
5

Intrusion Detection and Prevention Systems

3 hours
  • IDS/IPS Architecture and Types
    Network-based vs host-based, signature vs anomaly detection, and deployment strategies
  • Snort and Suricata Configuration
    Rule writing, signature development, and real-time intrusion detection setup Hands-on
  • Security Information and Event Management
    SIEM integration, log correlation, incident response automation, and threat intelligence
  • Machine Learning for Threat Detection
    Behavioral analysis, anomaly detection algorithms, and AI-powered security systems AI Security
  • Incident Response and Forensics
    Digital forensics, malware analysis, and coordinated incident response procedures Forensics
