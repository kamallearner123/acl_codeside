Course Curriculum

1

Process Management

6 hours
  • Linux Kernel Architecture
    Understanding kernel space vs user space, system calls, and kernel modules
  • Process Creation and Lifecycle
    fork(), exec(), wait() system calls, process states, and PID management Hands-on
  • CPU Scheduling Algorithms
    Round-robin, priority-based, CFS scheduler, and real-time scheduling Kernel
  • Threads and Multitasking
    POSIX threads, thread synchronization, and parallel processing concepts
  • Memory Management
    Virtual memory, paging, swapping, and memory allocation strategies
  • Process Monitoring and Control
    ps, top, htop, systemctl, and performance monitoring tools Hands-on
2

File Management

5 hours
  • Linux File System Hierarchy
    FHS standard, /bin, /etc, /var, /home structure and organization
  • File Systems and Storage
    ext4, XFS, Btrfs, ZFS comparison and implementation strategies Hands-on
  • Permissions and Access Control
    chmod, chown, umask, ACLs, and advanced security permissions
  • File Operations and Links
    Hard links, symbolic links, file manipulation, and directory operations
  • File System Navigation and Search
    find, locate, grep, sed, awk for efficient file management Hands-on
  • Backup and Recovery
    tar, rsync, snapshot management, and disaster recovery strategies
3

Inter-Process Communication (IPC)

4 hours
  • IPC Mechanisms Overview
    Pipes, FIFOs, message queues, shared memory, and sockets comparison
  • Pipes and Named Pipes
    Anonymous pipes, named pipes (FIFOs), and shell pipe operations Hands-on
  • Message Queues and Signals
    System V message queues, POSIX message queues, and signal handling
  • Shared Memory and Semaphores
    Shared memory segments, semaphore synchronization, and race condition prevention Advanced
  • Network IPC and Sockets
    Unix domain sockets, TCP/UDP sockets, and network communication patterns
  • IPC Programming Examples
    Practical C programming examples for all IPC mechanisms Hands-on
4

Linux System Administration

5 hours
  • System Boot Process and Init Systems
    GRUB bootloader, systemd, SysV init, and boot sequence analysis
  • User and Group Management
    useradd, usermod, passwd, sudo configuration, and security policies Admin
  • Service Management with Systemd
    Creating services, systemctl commands, service dependencies, and logging Hands-on
  • Network Configuration and Security
    iptables, firewalld, network interfaces, and security hardening
  • Package Management
    apt, yum, dnf, zypper package managers and software installation strategies
  • System Monitoring and Performance
    Log analysis, performance tuning, disk usage, and system health monitoring Admin
  • Security and Maintenance
    Security updates, vulnerability management, and system maintenance best practices
