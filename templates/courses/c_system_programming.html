Course Curriculum

1

Essentials of C Language (using GCC)

4 hours
  • C Language Fundamentals & GCC Compiler
    Variables, data types, operators, control structures, and mastering GCC compilation flags
  • Pointers and Memory Management
    Dynamic memory allocation, pointer arithmetic, and memory layout understanding
  • Functions and Advanced Data Structures
    Function pointers, structures, unions, and modular programming techniques
  • Build Systems and Makefiles
    Creating efficient build processes and managing large C projects Hands-on
2

Threads and Processes

5 hours
  • Process Creation and Management
    fork(), exec(), wait() system calls and process lifecycle management
  • POSIX Threads (pthreads)
    Thread creation, joining, detaching, and thread-specific data
  • Inter-Process Communication (IPC)
    Pipes, FIFOs, shared memory, and message queues Hands-on
  • Process Performance and Monitoring
    CPU scheduling, priority management, and system monitoring tools
3

Locks and Synchronization

4 hours
  • Mutex Locks and Critical Sections
    Thread synchronization, race conditions, and deadlock prevention
  • Condition Variables and Semaphores
    Advanced synchronization primitives and producer-consumer patterns
  • Read-Write Locks and Atomic Operations
    Optimizing concurrent access and lock-free programming techniques Advanced
  • Debugging Concurrent Programs
    Tools and techniques for finding race conditions and deadlocks
4

File Handling

4 hours
  • File I/O Operations
    Standard library vs system calls, buffering, and error handling
  • Directory Operations and File Metadata
    File permissions, timestamps, and directory traversal techniques
  • Asynchronous I/O and Performance
    Non-blocking I/O, select/poll/epoll, and high-performance file operations Advanced
  • Memory-Mapped Files
    mmap() system call and efficient large file processing
5

Debugging with GDB

4 hours
  • GDB Fundamentals
    Setting breakpoints, examining variables, and stepping through code
  • Advanced GDB Features
    Watchpoints, conditional breakpoints, and debugging core dumps Hands-on
  • Multi-threaded Debugging
    Thread-specific debugging and analyzing concurrent execution
  • GDB Scripting and Automation
    Custom GDB commands and automated debugging workflows
6

Memory Analysis with Valgrind

4 hours
  • Memory Error Detection
    Finding memory leaks, buffer overflows, and use-after-free errors
  • Performance Profiling with Callgrind
    CPU profiling, call graphs, and performance bottleneck identification Hands-on
  • Cache Performance Analysis
    Cachegrind for memory hierarchy optimization
  • Thread Safety Analysis
    Helgrind and DRD for detecting race conditions and synchronization issues
7

Unit Testing with Google Test

5 hours
  • Testing Framework Setup
    Installing Google Test, writing first tests, and test organization
  • Test Cases and Assertions
    EXPECT vs ASSERT macros, parameterized tests, and test fixtures Hands-on
  • Mocking and Test Doubles
    Google Mock framework for isolating units under test
  • Continuous Integration
    Integrating tests into build systems and CI/CD pipelines
