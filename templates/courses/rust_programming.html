Course Curriculum

1

Importance of Rust

3 hours
  • Industry News and Adoption Trends
    Microsoft's commitment to Rust, Linux kernel integration, and security-focused initiatives
  • Standards and Compliance
    ISO standardization efforts, safety-critical system requirements, and regulatory compliance
  • Security Vulnerabilities in C/C++
    Memory safety issues, buffer overflows, and how Rust prevents common security flaws
  • Performance Benchmarks
    Rust vs C/C++ performance comparison and zero-cost abstraction principles
2

Rust Syntax and Ownership

6 hours
  • Rust Fundamentals
    Variables, data types, functions, and basic syntax patterns
  • Ownership System
    Understanding ownership rules, move semantics, and memory management Core Concept
  • References and Borrowing
    Immutable and mutable references, borrowing rules, and lifetime annotations
  • Slices and String Handling
    String vs &str, array slices, and efficient text processing Hands-on

Memory-Safe System Design

5 hours
  • Memory Safety Guarantees
    Compile-time memory safety, preventing data races, and eliminating null pointer dereferences
  • Smart Pointers and RAII
    Box, Rc, Arc, RefCell, and resource management patterns Advanced
  • Lifetime Management
    Lifetime parameters, static lifetimes, and complex borrowing scenarios
  • Thread Safety by Design
    Send and Sync traits, data race prevention, and safe concurrency patterns Hands-on

Error Handling, Traits, Structs, Enums

5 hours
  • Robust Error Handling
    Result and Option types, error propagation with ?, and custom error types
  • Traits and Generics
    Defining traits, trait bounds, associated types, and generic programming Core Concept
  • Structs and Methods
    Struct definition, implementation blocks, associated functions, and data modeling
  • Enums and Pattern Matching
    Algebraic data types, match expressions, and exhaustive pattern matching Hands-on

Crates and Package Management

3 hours
  • Cargo Package Manager
    Creating projects, managing dependencies, and publishing crates
  • Module System
    Organizing code with modules, visibility rules, and project structure Hands-on
  • Popular Crates Ecosystem
    Essential crates: serde, tokio, clap, and ecosystem navigation
  • Testing and Documentation
    Unit tests, integration tests, doc tests, and benchmarking

Asynchronous Programming

5 hours
  • Async/Await Fundamentals
    Understanding futures, async functions, and the Rust async runtime
  • Tokio Runtime

  • Streams and Channels
    Async iterators, message passing, and concurrent data processing
  • Network Programming
    TCP/UDP sockets, HTTP clients, and async networking patterns

Actix and Web Libraries

4 hours
  • Actix Web Framework

Systems-Level Applications

  • Command-Line Applications
